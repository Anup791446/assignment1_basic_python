# Basic Operations
## Overview

Python program in basic_operations.py is a basic calculator that takes two numbers as input from the user and performs addition, subtraction, multiplication, and division. 
The results of each operation are then displayed on the screen.

## Features

*   **User Input:** Prompts the user to enter two numbers.
*   **Basic Operations:**  Performs addition, subtraction, multiplication, and division.
*   **Clear Output:**  Displays the result of each operation in a readable format.

## Getting Started

### Prerequisites

*   Python 3.x installed on your system.  You can download it from [https://www.python.org/downloads/](https://www.python.org/downloads/).

### Installation

1.  **Clone the repository :

    ```bash
    git clone <https://github.com/Anup791446/assignment1_basic_python>
    ```

2.  **Navigate to the project directory:**

    ```bash
    cd <your_project_directory>
    ```

### Usage

1.  **Run the program:**

    ```bash
    python basic_operations.py  
    ```

2.  **Follow the prompts:** The program will ask you to enter two numbers.  Enter the numbers and press Enter after each.

3.  **View the results:** The program will display the results of the addition, subtraction, multiplication, and division operations.

# Name Greeter Program

## Description

This program takes a user's first name and last name as input, concatenates them to form a full name, 
and then prints a personalized greeting message. It's a simple demonstration of user input, string manipulation, and output.

## Features
